### Detailed Examples

#### 1. **Handling Events:**

Let’s start by creating a simple button click event handler:

```jsx
import React, { useState } from 'react';

function ToggleButton() {
  const [isOn, setIsOn] = useState(false);

  const handleClick = () => {
    setIsOn(!isOn);
  };

  return (
    <button onClick={handleClick}>
      {isOn ? 'ON' : 'OFF'}
    </button>
  );
}

export default ToggleButton;
```

- **Explanation:** 
  - The `useState` hook initializes a state variable `isOn` to track whether the button is "ON" or "OFF".
  - The `handleClick` function toggles the `isOn` state between `true` and `false`.
  - The button’s text is conditionally rendered based on the `isOn` state.

#### 2. **Conditional Rendering:**

Let’s create a component that conditionally renders a greeting message based on user login status:

```jsx
import React, { useState } from 'react';

function Greeting() {
  const [isLoggedIn, setIsLoggedIn] = useState(false);

  const handleLogin = () => {
    setIsLoggedIn(true);
  };

  const handleLogout = () => {
    setIsLoggedIn(false);
  };

  return (
    <div>
      {isLoggedIn ? (
        <h1>Welcome back!</h1>
      ) : (
        <h1>Please sign in.</h1>
      )}
      <button onClick={isLoggedIn ? handleLogout : handleLogin}>
        {isLoggedIn ? 'Logout' : 'Login'}
      </button>
    </div>
  );
}

export default Greeting;
```

- **Explanation:**
  - The `Greeting` component manages a boolean state `isLoggedIn`.
  - Based on the `isLoggedIn` state, it conditionally renders either "Welcome back!" or "Please sign in."
  - The button toggles between "Login" and "Logout" based on the user’s state.

#### 3. **Handling Form Events:**

Create a simple form that handles user input:

```jsx
import React, { useState } from 'react';

function LoginForm() {
  const [username, setUsername] = useState('');
  const [password, setPassword] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    alert(`Logging in as: ${username}`);
  };

  return (
    <form onSubmit={handleSubmit}>
      <div>
        <label>
          Username:
          <input 
            type="text" 
            value={username} 
            onChange={(e) => setUsername(e.target.value)} 
          />
        </label>
      </div>
      <div>
        <label>
          Password:
          <input 
            type="password" 
            value={password} 
            onChange={(e) => setPassword(e.target.value)} 
          />
        </label>
      </div>
      <button type="submit">Login</button>
    </form>
  );
}

export default LoginForm;
```

- **Explanation:**
  - The `LoginForm` component handles user input for username and password.
  - The form submission is handled by `handleSubmit`, which prevents the default form behavior and displays an alert with the entered username.
